💰 Finova – Smart Personal Finance & Expense Tracking App

🚀 Overview
Finova is a modern mobile finance management application designed to help users track expenses, monitor spending habits, manage budgets, and achieve financial goals. The project focuses on creating an intuitive and visually appealing user experience through a clean and user-centered UI/UX design approach.

---

✨ Features

🔐 User Authentication
- Secure Login & Registration
- Google Sign-In Support
- Biometric Authentication (Fingerprint / Face ID)

📊 Dashboard
- Real-time Account Balance Overview
- Income & Expense Summary
- Recent Transactions
- Monthly Spending Insights

💸 Expense Management
- Add New Expenses
- Categorize Transactions
- Expense Notes & Date Tracking
- Budget Monitoring

📈 Analytics & Reports
- Spending Distribution Charts
- Monthly Expense Analysis
- Savings Tracking
- Financial Performance Insights

👛 Wallet Management
- Multiple Account Support
- Transaction History
- Quick Financial Actions

⚙️ Profile & Settings
- User Profile Management
- Notification Preferences
- Dark Mode Support
- Security Settings

🎨 Design Process
The project follows a structured UI/UX workflow:
1️⃣ User Research
2️⃣ Problem Definition
3️⃣ User Flow Design
4️⃣ Wireframing
5️⃣ High-Fidelity UI Design
6️⃣ Interactive Prototyping
7️⃣ Usability Testing & Improvements
